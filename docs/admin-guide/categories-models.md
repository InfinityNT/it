# Categories & Device Models

Device categories and models define the types of devices tracked in the system.

## Categories

Categories are broad device types such as:

- Laptops
- Monitors
- Phones
- Tablets
- Printers
- Networking Equipment

### Managing Categories

1. Open **Manage Categories** from the Devices page
2. Add new categories with a name and icon
3. Edit or deactivate existing categories
4. Each category can have custom specification fields

### Category Specifications

Categories can define specification templates — fields like "RAM", "Storage", "Screen Size" that appear automatically when adding devices of that type.

## Device Models

Device models represent specific products within a category (e.g., "Dell Latitude 5540" under "Laptops").

### Managing Models

1. Open **Manage Device Models** from the Devices page
2. Add a model with manufacturer, model name, and category
3. Edit existing models as needed

### Model Information

Each device model stores:

- **Manufacturer** — The brand (Dell, Apple, HP, etc.)
- **Model Name** — The specific product name
- **Category** — Which device category it belongs to
