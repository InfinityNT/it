# Locations

Locations represent physical sites or buildings where devices are stored or used.

## Viewing Locations

Locations are visible when adding or editing devices. To manage all locations, open the location management modal from the device form or settings.

## Adding a Location

1. Open the **Manage Locations** dialog
2. Click **Add Location**
3. Enter the location name and code (e.g., "Main Office" / "HQ")
4. Save the new location

## Editing a Location

1. Open **Manage Locations**
2. Click the edit icon next to the location
3. Update the name, code, or active status
4. Save changes

## Deactivating a Location

Locations can be deactivated instead of deleted. Deactivated locations won't appear in dropdown menus but existing device records referencing them are preserved.

## Location Codes

Each location has a short code (e.g., "HQ", "DC1", "WH-A") used for quick identification in lists and reports. Codes must be unique across the system.
