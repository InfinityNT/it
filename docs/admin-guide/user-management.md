# User Management

Administrators can create, edit, and manage user accounts from the Users page.

## Creating a User

1. Navigate to the **Users** page
2. Click **Add User**
3. Fill in the required fields:
   - Username
   - First and last name
   - Email address
   - Password (or generate a temporary one)
4. Assign a role/permission group
5. Optionally link to an employee record

## Roles and Permissions

The system uses permission groups to control access:

- **System Manager** — Full access to all features including system settings
- **Device Manager** — Can add, edit, and manage devices and assignments
- **Viewer** — Read-only access to devices and reports
- **User** — Basic access with limited visibility

## Deactivating a User

To deactivate a user account without deleting it:

1. Open the user's detail view
2. Click **Deactivate**
3. Confirm the action

Deactivated users cannot log in but their records and audit history are preserved.

## Resetting a Password

Administrators can reset any user's password:

1. Open the user's detail view
2. Click **Reset Password**
3. A temporary password is generated
4. Share it with the user — they'll be required to change it on next login

## Reactivating a User

Previously deactivated users can be reactivated from their detail view using the **Reactivate** button.
