# Dashboard

The dashboard is your home screen after logging in. It provides a quick overview of the system status and recent activity.

## Stats Cards

At the top of the dashboard you'll see summary cards showing:

- **Total Devices** — The total number of devices in the system
- **Available Devices** — Devices not currently assigned to anyone
- **Assigned Devices** — Devices currently assigned to employees
- **Employees** — Total number of employees in the system

## Recent Activity

The activity feed shows the most recent actions taken in the system, including:

- Device additions and modifications
- Assignment and return operations
- Employee record changes
- Approval status updates

## Quick Actions

The sidebar contains configurable quick action buttons for frequently used operations. You can customize which actions appear in your sidebar from the Settings page.

## Navigation

Use the top navigation bar to move between sections. On mobile devices, tap the menu icon to access navigation links and quick actions in a slide-out drawer.
