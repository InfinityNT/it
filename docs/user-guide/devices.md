# Devices

The Devices section lets you browse, search, and view details for all IT assets tracked in the system.

## Device List

The main device list shows all registered devices with key information:

- **Asset Tag** — Unique identifier for the device
- **Category** — The device type (laptop, monitor, phone, etc.)
- **Model** — Manufacturer and model name
- **Status** — Current status (Available, Assigned, Maintenance, Retired)
- **Assigned To** — The employee currently using the device, if any

## Filtering and Search

Use the filter bar above the device list to narrow results by:

- **Status** — Filter by device status
- **Category** — Filter by device category
- **Location** — Filter by physical location
- **Search** — Free-text search across asset tags, serial numbers, and model names

## Device Details

Click on any device to open its detail view, which shows:

- Full specifications and metadata
- Assignment history
- Current assignment status
- Notes and maintenance records

## Adding a Device

If you have the appropriate permissions, use the "Add Device" button to register a new device. You'll need to provide:

1. Select or create a device model
2. Enter the asset tag and serial number
3. Set the initial status and location
4. Add any additional notes
