# Reports

The Reports section provides built-in analytics and the ability to create custom reports.

## Built-in Reports

Several pre-configured reports are available:

- **Device Inventory** — Full device list with specs, status, and location data
- **Assignment History** — All assignments, returns, and transfers over time
- **Utilization Analysis** — Usage patterns and resource optimization insights
- **Maintenance** — Service schedules, warranty status, and device health
- **Cost Analysis** — Total ownership costs and depreciation
- **User Activity** — Employee usage patterns and equipment preferences

## Charts

The reports page includes interactive charts:

- **Device Distribution** — Breakdown by category (pie chart)
- **Status Overview** — Current device status distribution
- **Assignment Trends** — Monthly assignment activity over the last 12 months
- **Top Users** — Employees with the most assigned devices

## Custom Reports

Click "Create Custom Report" to build a report with:

1. Choose data sources (Devices, Employees, Assignments, Approvals)
2. Select which columns to include
3. Apply filters and sorting
4. Export in CSV, PDF, or JSON format

## Exporting

All reports can be exported. Click the export button on any chart or use "Export All" to download everything at once. Available formats include CSV, PDF, and JSON.
