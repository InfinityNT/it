# Search

The platform provides multiple ways to find devices, employees, and other records.

## Global Search

The global search bar in the navigation allows you to quickly find items across the entire system. Type a query to search across:

- Device asset tags and serial numbers
- Device model names and manufacturers
- Employee names and email addresses

Results are grouped by type and shown in a dropdown. Click a result to navigate directly to its detail view.

## Advanced Search

The dedicated Search page provides more powerful filtering:

- **Category filters** — Narrow results to specific device categories
- **Status filters** — Search within specific statuses
- **Date ranges** — Find items created or modified within a time period
- **Location** — Filter by physical location
- **Combined filters** — Stack multiple filters for precise results

## Tips

- Use partial matches — searching "Mac" will find "MacBook Pro", "Mac Mini", etc.
- Asset tags and serial numbers are matched exactly when possible
- Search is case-insensitive
- Clear filters to return to the full list view
